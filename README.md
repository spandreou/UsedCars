A PHP & MySQL web application for managing used car listings.

Tech Stack:
- PHP
- MySQL
- Bootstrap 5
- JavaScript
- particles.js

Setup:
1. Import sql/webcars.sql into MySQL
2. Configure database credentials in includes/db.php
3. Run the project via Apache (XAMPP)
